# Application Setup

Prerequisites
├── .Net Framwork: 10.0
└── c#: 14

Clone git repository, Build and Start application
├── Create any directory and open command prompt in same directory
├── cmd> git clone  https://github.com/rajeevsharma4nagarro/Design-Patterns-POC-Basic.git
├── cmd> dotnet build
├── cmd> dotnet run
